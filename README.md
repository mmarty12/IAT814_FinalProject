# IAT814_FinalProject
Final project for the IAT814 (IAT355) "Introduction to Visual Analytics" course.

---
My project is designed to help users quickly understand patterns in board games through interactive data visualizations. It brings together key attributes such as user ratings, number of votes, genres, mechanics, playtime, and release year. Together, these dimensions allow users (primarily casual and family players) to explore not just individual games, but also broader trends across the board game world. The project presents 10 sections, describing the top games by each attribute and investigating deeper correlations between them. 

The visualizations are based on a large [dataset](https://www.kaggle.com/datasets/sujaykapadnis/board-games) of approximately 90,000 board games, originally sourced from BoardGameGeek and found on Kaggle.
